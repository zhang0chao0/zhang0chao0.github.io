分支

一般开发会用dev分支，而不是master分支，因为master分支会直接对接线上的代码，安全性极高，不会让人直接修改，我开发都是在dev分支上修改，提交代码后，再由其它人员（权限较高，一般是代码负责人）合并到master分支。
