зұ»еҠ иҪҪ

еҸ‘иЎЁдәҺ 2017-07-30 | еҲҶзұ»дәҺ Java |

еүҚиЁҖ

иҰҒзҗҶи§ЈRTTIпјҲиҝҗиЎҢж—¶зұ»еһӢдҝЎжҒҜпјүеңЁJavaдёӯзҡ„е·ҘдҪңеҺҹзҗҶпјҢеҝ…йЎ»зҹҘйҒ“зұ»еһӢдҝЎжҒҜжҳҜеҰӮдҪ•иЎЁзӨәзҡ„гҖӮ

зұ»еҠ иҪҪжңәеҲ¶

жҜҸдёӘзұ»йғҪжңүдёҖдёӘClassеҜ№иұЎпјҢжҜҸеҪ“зј–иҜ‘дәҶдёҖдёӘзұ»пјҢе°ұдјҡдә§з”ҹдёҖдёӘClassеҜ№иұЎпјҢдҝқеӯҳеңЁеҗҢеҗҚзҡ„.classж–Үд»¶дёӯгҖӮ
еҪ“зЁӢеәҸиҝҗиЎҢж—¶пјҢиҷҡжӢҹжңәжҠҠжҸҸиҝ°зұ»зҡ„ж•°жҚ®д»Һclassж–Үд»¶дёӯеҠ иҪҪеҲ°еҶ…еӯҳпјҢе№¶еҜ№ж•°жҚ®иҝӣиЎҢж ЎйӘҢпјҢиҪ¬жҚўи§Јжһҗе’ҢеҲқе§ӢеҢ–пјҢжңҖз»ҲеҪўжҲҗеҸҜд»Ҙиў«иҷҡжӢҹжңәзӣҙжҺҘдҪҝз”Ёзҡ„Javaзұ»еһӢпјҢиҝҷе°ұжҳҜиҷҡжӢҹжңәзҡ„зұ»еҠ иҪҪжңәеҲ¶гҖӮ

Javaдёӯзҡ„зұ»еҠ иҪҪеҷЁеӨ§иҮҙеҸҜд»ҘеҲҶжҲҗдёӨзұ»пјҢдёҖзұ»жҳҜзі»з»ҹжҸҗдҫӣзҡ„пјҢеҸҰеӨ–дёҖзұ»еҲҷжҳҜз”ұ Java еә”з”ЁејҖеҸ‘дәәе‘ҳзј–еҶҷзҡ„пјҢд»ҘдёӢзҹҘиҜҶзӮ№жҖ»з»“дәҺзҪ‘з»ңгҖӮ

  • еј•еҜјзұ»еҠ иҪҪеҷЁпјҲbootstrap class loaderпјүпјҡ
    е®ғз”ЁжқҘеҠ иҪҪ Javaзҡ„ж ёеҝғеә“(jre/lib/rt.jar)пјҢжҳҜз”ЁеҺҹз”ҹC++д»Јз ҒжқҘе®һзҺ°зҡ„пјҢе№¶дёҚз»§жүҝиҮӘjava.lang.ClassLoaderгҖӮ
  • жү©еұ•зұ»еҠ иҪҪеҷЁпјҲextensions class loaderпјүпјҡ
    е®ғз”ЁжқҘеҠ иҪҪ Javaзҡ„жү©еұ•еә“(jre/ext/*.jar)гҖӮJavaиҷҡжӢҹжңәзҡ„е®һзҺ°дјҡжҸҗдҫӣдёҖдёӘжү©еұ•еә“зӣ®еҪ•гҖӮиҜҘзұ»еҠ иҪҪеҷЁеңЁжӯӨзӣ®еҪ•йҮҢйқўжҹҘжүҫе№¶еҠ иҪҪ Java зұ»гҖӮ
  • зі»з»ҹзұ»еҠ иҪҪеҷЁпјҲsystem class loaderпјүпјҡ
    е®ғж №жҚ® Java еә”з”Ёзҡ„зұ»и·Ҝеҫ„пјҲCLASSPATHпјүжқҘеҠ иҪҪ Java зұ»гҖӮдёҖиҲ¬жқҘиҜҙпјҢJavaеә”з”Ёзҡ„зұ»йғҪжҳҜз”ұе®ғжқҘе®ҢжҲҗеҠ иҪҪзҡ„гҖӮеҸҜд»ҘйҖҡиҝҮ ClassLoader.getSystemClassLoader()жқҘиҺ·еҸ–е®ғгҖӮ
  • иҮӘе®ҡд№үзұ»еҠ иҪҪеҷЁпјҲcustom class loaderпјүпјҡ
    йҷӨдәҶзі»з»ҹжҸҗдҫӣзҡ„зұ»еҠ иҪҪеҷЁд»ҘеӨ–пјҢејҖеҸ‘дәәе‘ҳеҸҜд»ҘйҖҡиҝҮз»§жүҝjava.lang.ClassLoaderзұ»зҡ„ж–№ејҸе®һзҺ°иҮӘе·ұзҡ„зұ»еҠ иҪҪеҷЁпјҢд»Ҙж»Ўи¶ідёҖдәӣзү№ж®Ҡзҡ„йңҖжұӮгҖӮ

дёүдёӘжңәеҲ¶

  • е§”жүҳжңәеҲ¶пјҡ
    жҳҜжҢҮе°ҶеҠ иҪҪдёҖдёӘзұ»зҡ„иҜ·жұӮдәӨз»ҷзҲ¶зұ»еҠ иҪҪеҷЁпјҢеҰӮжһңиҝҷдёӘзҲ¶зұ»еҠ иҪҪеҷЁдёҚиғҪеӨҹжүҫеҲ°жҲ–иҖ…еҠ иҪҪиҝҷдёӘзұ»пјҢйӮЈд№ҲеҶҚеҠ иҪҪе®ғгҖӮ
  • еҸҜи§ҒжҖ§зҡ„еҺҹзҗҶпјҡ
    жҳҜеӯҗзұ»зҡ„еҠ иҪҪеҷЁеҸҜд»ҘзңӢи§ҒжүҖжңүзҡ„зҲ¶зұ»еҠ иҪҪеҷЁеҠ иҪҪзҡ„зұ»пјҢиҖҢзҲ¶зұ»еҠ иҪҪеҷЁзңӢдёҚеҲ°еӯҗзұ»еҠ иҪҪеҷЁеҠ иҪҪзҡ„зұ»гҖӮ
  • еҚ•дёҖжҖ§еҺҹзҗҶпјҡ
    жҳҜжҢҮд»…еҠ иҪҪдёҖдёӘзұ»дёҖж¬ЎпјҢиҝҷжҳҜз”ұе§”жүҳжңәеҲ¶зЎ®дҝқеӯҗзұ»еҠ иҪҪеҷЁдёҚдјҡеҶҚж¬ЎеҠ иҪҪзҲ¶зұ»еҠ иҪҪеҷЁеҠ иҪҪиҝҮзҡ„зұ»гҖӮ

зұ»еҠ иҪҪиҝҮзЁӢеӣҫ



жӯҘйӘӨ

  1. иЈ…иҪҪпјҡ
    жҹҘжүҫе№¶еҠ иҪҪзұ»зҡ„дәҢиҝӣеҲ¶ж•°жҚ®гҖӮ
  2. й“ҫжҺҘпјҡ
    йӘҢиҜҒпјҡзЎ®дҝқиў«еҠ иҪҪзұ»дҝЎжҒҜз¬ҰеҗҲJVMи§„иҢғгҖҒжІЎжңүе®үе…Ёж–№йқўзҡ„й—®йўҳгҖӮ
    еҮҶеӨҮпјҡдёәзұ»зҡ„йқҷжҖҒеҸҳйҮҸеҲҶй…ҚеҶ…еӯҳпјҢе№¶е°Ҷе…¶еҲқе§ӢеҢ–дёәй»ҳи®ӨеҖјгҖӮ
    и§ЈжһҗпјҡжҠҠиҷҡжӢҹжңәеёёйҮҸжұ пјҲжҢҮжҜҸдёӘзұ»з§Ғжңүзҡ„еӯҳж”ҫзұ»еҗҚгҖҒж–№жі•еҗҚе’Ңз¬ҰеҸ·зҡ„йӣҶеҗҲпјүдёӯзҡ„з¬ҰеҸ·еј•з”ЁиҪ¬жҚўдёәзӣҙжҺҘеј•з”ЁгҖӮ
  3. еҲқе§ӢеҢ–пјҡ
    дёәзұ»зҡ„йқҷжҖҒеҸҳйҮҸиөӢдәҲжӯЈзЎ®зҡ„еҲқе§ӢеҖјгҖӮ
зұ»зҡ„еҲқе§ӢеҢ–жӯҘйӘӨпјҡ
  1. еҰӮжһңиҝҷдёӘзұ»иҝҳжІЎжңүиў«еҠ иҪҪе’Ңй“ҫжҺҘпјҢйӮЈе…ҲиҝӣиЎҢеҠ иҪҪе’Ңй“ҫжҺҘгҖӮ
  2. еҒҮеҰӮиҝҷдёӘзұ»еӯҳеңЁзӣҙжҺҘзҲ¶зұ»пјҢе№¶дё”иҝҷдёӘзұ»иҝҳжІЎжңүиў«еҲқе§ӢеҢ–пјҲжіЁж„ҸпјҡеңЁдёҖдёӘзұ»еҠ иҪҪеҷЁдёӯпјҢзұ»еҸӘиғҪеҲқе§ӢеҢ–дёҖж¬ЎпјүпјҢйӮЈе°ұеҲқе§ӢеҢ–зӣҙжҺҘзҡ„зҲ¶зұ»пјҲдёҚйҖӮз”ЁдәҺжҺҘеҸЈпјүгҖӮ
  3. еҰӮжһңзұ»дёӯеӯҳеңЁstaticж ҮиҜҶзҡ„еқ—пјҢйӮЈе°ұдҫқж¬Ўжү§иЎҢиҝҷдәӣеҲқе§ӢеҢ–иҜӯеҸҘгҖӮ

зұ»еҠ иҪҪеҷЁзҡ„е·ҘдҪңеҺҹзҗҶ

е§”жүҳжңәеҲ¶

еҪ“дёҖдёӘзұ»еҠ иҪҪе’ҢеҲқе§ӢеҢ–зҡ„ж—¶еҖҷпјҢзұ»д»…еңЁжңүйңҖиҰҒеҠ иҪҪзҡ„ж—¶еҖҷиў«еҠ иҪҪгҖӮеҒҮи®ҫдҪ жңүдёҖдёӘеә”з”ЁйңҖиҰҒзҡ„зұ»еҸ«дҪңAbc.classпјҢйҰ–е…ҲеҠ иҪҪиҝҷдёӘзұ»зҡ„иҜ·жұӮз”ұApplicationзұ»еҠ иҪҪеҷЁе§”жүҳз»ҷе®ғзҡ„зҲ¶зұ»еҠ иҪҪеҷЁExtensionзұ»еҠ иҪҪеҷЁпјҢз„¶еҗҺеҶҚе§”жүҳз»ҷBootstrapзұ»еҠ иҪҪеҷЁгҖӮBootstrapзұ»еҠ иҪҪеҷЁдјҡе…ҲзңӢзңӢrt.jarдёӯжңүжІЎжңүиҝҷдёӘзұ»пјҢеӣ дёәе№¶жІЎжңүиҝҷдёӘзұ»пјҢжүҖд»ҘиҝҷдёӘиҜ·жұӮз”ұеӣһеҲ°Extensionзұ»еҠ иҪҪеҷЁпјҢе®ғдјҡжҹҘзңӢjre/lib/extзӣ®еҪ•дёӢжңүжІЎжңүиҝҷдёӘзұ»пјҢеҰӮжһңиҝҷдёӘзұ»иў«Extensionзұ»еҠ иҪҪеҷЁжүҫеҲ°дәҶпјҢйӮЈд№Ҳе®ғе°Ҷиў«еҠ иҪҪпјҢиҖҢApplicationзұ»еҠ иҪҪеҷЁдёҚдјҡеҠ иҪҪиҝҷдёӘзұ»пјӣиҖҢеҰӮжһңиҝҷдёӘзұ»жІЎжңүиў«Extensionзұ»еҠ иҪҪеҷЁжүҫеҲ°пјҢйӮЈд№ҲеҶҚз”ұApplicationзұ»еҠ иҪҪеҷЁд»ҺclasspathдёӯеҜ»жүҫгҖӮи®°дҪҸclasspathе®ҡд№үзҡ„жҳҜзұ»ж–Үд»¶зҡ„еҠ иҪҪзӣ®еҪ•пјҢиҖҢPATHжҳҜе®ҡд№үзҡ„жҳҜеҸҜжү§иЎҢзЁӢеәҸеҰӮjavacпјҢjavaзӯүзҡ„жү§иЎҢи·Ҝеҫ„гҖӮ

еҸҜи§ҒжҖ§жңәеҲ¶

ж №жҚ®еҸҜи§ҒжҖ§жңәеҲ¶пјҢеӯҗзұ»еҠ иҪҪеҷЁеҸҜд»ҘзңӢеҲ°зҲ¶зұ»еҠ иҪҪеҷЁеҠ иҪҪзҡ„зұ»пјҢиҖҢеҸҚд№ӢеҲҷдёҚиЎҢгҖӮжүҖд»ҘдёӢйқўзҡ„дҫӢеӯҗдёӯпјҢеҪ“Abc.classе·Із»Ҹиў«Applicationзұ»еҠ иҪҪеҷЁеҠ иҪҪиҝҮдәҶпјҢз„¶еҗҺеҰӮжһңжғіиҰҒдҪҝз”ЁExtensionзұ»еҠ иҪҪеҷЁеҠ иҪҪиҝҷдёӘзұ»пјҢе°ҶдјҡжҠӣеҮәjava.lang.ClassNotFoundExceptionејӮеёёгҖӮ

еҚ•дёҖжҖ§жңәеҲ¶

ж №жҚ®иҝҷдёӘжңәеҲ¶пјҢзҲ¶еҠ иҪҪеҷЁеҠ иҪҪиҝҮзҡ„зұ»дёҚиғҪиў«еӯҗеҠ иҪҪеҷЁеҠ иҪҪз¬¬дәҢж¬ЎгҖӮиҷҪз„¶йҮҚеҶҷиҝқеҸҚе§”жүҳе’ҢеҚ•дёҖжҖ§жңәеҲ¶зҡ„зұ»еҠ иҪҪеҷЁжҳҜеҸҜиғҪзҡ„пјҢдҪҶиҝҷж ·еҒҡе№¶дёҚеҸҜеҸ–гҖӮдҪ еҶҷиҮӘе·ұзҡ„зұ»еҠ иҪҪеҷЁзҡ„ж—¶еҖҷеә”иҜҘдёҘж јйҒөе®ҲиҝҷдёүжқЎжңәеҲ¶гҖӮ

ClassLoaderзұ»

java.lang.ClassLoaderзұ»зҡ„еҹәжң¬иҒҢиҙЈе°ұжҳҜж №жҚ®дёҖдёӘжҢҮе®ҡзҡ„зұ»зҡ„еҗҚз§°пјҢжүҫеҲ°жҲ–иҖ…з”ҹжҲҗе…¶еҜ№еә”зҡ„еӯ—иҠӮд»Јз ҒпјҢз„¶еҗҺд»Һиҝҷдәӣеӯ—иҠӮд»Јз Ғдёӯе®ҡд№үеҮәдёҖдёӘJavaзұ»пјҢеҚіjava.lang.Classзұ»зҡ„дёҖдёӘе®һдҫӢгҖӮ

ClassLoaderжҸҗдҫӣдәҶдёҖзі»еҲ—зҡ„ж–№жі•пјҢжҜ”иҫғйҮҚиҰҒзҡ„ж–№жі•еҰӮпјҡ
