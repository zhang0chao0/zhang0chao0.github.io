HTTP协议

发表于 2017-07-23 | 分类于 HTTP

前言

谈到HTTP协议，就不得不提谷歌浏览器的一个插件postman，无论你是做前端还是做后端的，这个工具都能发挥作用。建议程序员小伙伴安装使用（谷歌浏览器程序员必备就不用说了吧）。



我通过简单的GET、POST请求为大家讲解：

GET请求



这是我本地的一个GET请求：localhost:3000/adcreate/application?application_id=1497002130351668，通过code可以方便的查看HTTP代码，包括各种语言的请求代码都能生成（postman的好用之处）。
