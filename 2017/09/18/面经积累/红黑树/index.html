红黑树

红黑树是一颗平衡二叉树，图示：



规则
  1. 节点只能为红色或者黑色。
  2. 根节点和叶节点必须为黑色。（这里的叶子节点并不是普通的叶子节点，而是人为的补了NULL，以NULL作为叶子节点）
  3. 红色节点的子节点为黑色。
  4. 任意一节点到达叶子节点（只能顺着走最短路，不能绕弯）所经过的黑色节点数目相同。（例如8号，到达包括NULL叶子节点所经过的黑色节点都是2个）
插入操作

如果插入黑色，会破坏最重要的4条件，所以只能插入红色节点，但是插入后需要调整或者变色。

情景1：父节点为黑色，直接插入

例如插入1的左子树，这样完全符合规则，不用改变。

情景2：父节点为红色，父节点兄弟节点对称

G为黑色，P、U为红色，此时要插入红色的N：



插入节点N，此时需要递归的把P、U、G都变为相反色，来确保符合规则。最后把根节点置为黑色。

情景3：父节点红色，父节点兄弟节点不对称


例如：P、U不同色，P红U黑，此时插入N节点，把P和G置为相反色，然后PG以P为中心点旋转。
